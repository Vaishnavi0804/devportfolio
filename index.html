Projects

A Deep Learning Multi-Class Time Series Approach for Human Activity Recognition

This project explores time series classification for activity recognition using accelerometer data from 15 subjects performing various tasks. By leveraging deep learning multi-class time series classification algorithms, the goal is to improve upon existing methods that rely on single-point ensemble classification. Unlike the more complex UCI HAR dataset, which includes gyroscope data and 560 features, this dataset focuses on simpler accelerometer data (x, y, z axes). The project evaluates whether capturing temporal coherence with deep learning provides significant advantages in accuracy and efficiency over single-point approaches, with benchmarks for comparison. This concise description highlights the essence of the project while maintaining its professional tone.
